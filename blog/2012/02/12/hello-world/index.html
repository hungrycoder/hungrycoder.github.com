Hello World

Feb 12th, 2012

I am trying out blogging with OctoPress. Here are my initial thoughts.

Some things I like:

  1. It’s all static! – Can’t really beat this. Edit posts in your favorite text edit and publish when you are ready.
  2. Sass for CSS
  3. Decent docs
  4. Integrated version control
  5. Simple deploying

Some things I don’t like:

  1. Lots of dependencies (almost 30 by my count).
  2. It took about 5 minutes to bootstrap to Ruby 1.9.2.
  3. Rake for everything. Can’t I just add a post file?
  4. Posts in markdown

Here’s a Jekyll plugin for Restructured Text instead of Markdown. To install the jekyll-rst plugin use:

$ pip install pygments
$ pip install docutils
$ cd <octopress-project-path>
$ git submodule add https://github.com/xdissent/jekyll-rst.git plugins/jekyll-rst
